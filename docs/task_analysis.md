# Task Analysis

A structured breakdown of the tasks and analytical steps required to answer the research question.

> **Research question:** "How do changes in gender inequality relate to cross-country differences in female incarceration in Europe?"

---

## 1. Domain Questions

_What does the user / audience need to know?_

- [ ] Which EU countries have the highest / lowest gender equality scores?
- [ ] Which EU countries have the highest / lowest female incarceration rates?
- [ ] Is there a statistically observable association between GEI score and female incarceration rate across countries?
- [ ] Has this relationship changed over time?
- [ ] Are there regional patterns (Northern / Eastern / Central-Southern Europe)?
- [ ] Which sub-dimensions of gender equality (work, power, health, etc.) correlate most strongly with incarceration?
