# Gender Inequality Index vs Female Jail Population

## Requirements

Install dependencies with:
​```bash
pip install -r requirements.txt
​```

## Project Structure
├── data/
│   ├── Data to export/       # CSV outputs generated by the notebooks
│   ├── Gender index data/    # Source PDFs and Excel files for GII data
│   └── prison data/          # Source CSV with prison population data
├── notebooks/
│   ├── Gender_index_graph.ipynb
│   └── Prison_data_graph.ipynb
├── website/                  # Final HTML visualization


## How to Run

Open and run the notebooks **in this order**:

1. `notebooks/Gender_index_graph.ipynb`
2. `notebooks/Prison_data_graph.ipynb`

Each notebook can be run via Jupyter:
​```bash
jupyter notebook
​```

Running the notebooks will automatically generate the output CSV files
in `data/Data to export/`.
